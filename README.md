# Saman Traders - Retail Shop Website

Welcome to the official website project for **Saman Traders**, a premier retail shop in Kuliyapitiya offering toys, baby items, and daily essentials.

## Project Structure

- **index.html**: The main landing page containing the Hero section, Product Categories, About Us, and Contact information.
- **script.js**: JavaScript file handling mobile menu toggling, sticky navigation, and scroll animations.
- **assets/images/**: Contains SVG placeholder images for the website sections.

## Technologies Used

- **HTML5**: Semantic markup for structure.
- **Tailwind CSS**: Utility-first CSS framework for styling (via CDN).
- **JavaScript (Vanilla)**: For interactivity and DOM manipulation.
- **Google Fonts**: Inter and Poppins fonts for typography.

## How to Run

Simply open the `index.html` file in any modern web browser (Chrome, Edge, Firefox, etc.). No installation or server required.

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Modern UI**: Clean layout with glassmorphism effects and smooth animations.
- **Navigation**: Sticky navbar with mobile menu support.
- **Sections**:
  - **Hero**: Welcoming banner with shop highlights.
  - **Categories**: Showcase for Toys, Baby Items, and Retail Goods.
  - **Features**: Key selling points like Quality, Fresh Stock, etc.
  - **About**: Information about the shop's history.
  - **Contact**: Address, phone, and links.

## Customization

To update the content, edit the `index.html` file.
To change the images, replace the `.svg` files in `assets/images/` with your own images (JPG/PNG).
